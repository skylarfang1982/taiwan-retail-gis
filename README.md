# 📍 全台零售業展店與商圈數據決策系統 (Taiwan Retail GIS)

這是一套基於 **Python Streamlit** 核心開發的數據驅動展店決策系統，專為台灣零售業、餐飲業或高客單價品牌（如按摩椅專櫃）設計。透過視覺化地圖與權重演算法，協助管理階層進行精準的展店與商場進駐評估。

## 🌟 核心功能
1. **高收入熱點篩選**：結合財政部村里所得大數據，秒抓全台含金量最高、高消費力的黃金區域。
2. **目標客群精準媒合**：串接內政部人口結構，過濾出目標年齡層（35-55歲）最密集的藍海商圈。
3. **互動式 GIS 地圖**：使用 Plotly 打造動態地圖，直觀呈現最佳進駐百貨商場推薦與 ROI 財務模擬。
